# Collaborative Document Editor PoC

This proof of concept validates a minimal frontend-to-backend architecture for a collaborative document editor assignment.

## Scope

- React frontend calling a Python FastAPI backend
- Explicit shared contracts documented in `shared/contracts.md`
- In-memory document storage only
- Manual document creation, loading, and saving

## Intentionally Out of Scope

- Authentication and authorization
- Real-time collaboration and sync
- AI assistance
- Persistent database storage
- Version history
